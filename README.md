# 择日学技能

> "择日而用，趋吉避凶。" —— 传统术数

择日学是中国传统术数中的重要分支，用于选择吉日良辰进行婚嫁、开业、动土、出行等重要活动。本技能提供完整的择日查询和吉凶分析功能。

## ✨ 核心功能

- ✨ 建除十二神查询（建/除/满/平/定/执/破/危/成/收/开/闭）
- ✨ 黄道黑道查询（青龙/明堂/天刑/朱雀/金匮/天德/白虎/玉堂/天牢/玄武/司命/勾陈）
- ✨ 神煞吉凶查询（天乙贵人/文昌/桃花/驿马等）
- ✨ 宜忌事项查询
- ✨ 冲煞分析（冲生肖/煞方位）
- ✨ 择日建议（婚嫁/开业/动土/出行等）
- ✨ 纯 JavaScript 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确建除十二神计算
- 🔧 完整黄道黑道算法

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/ze-ri-skill.git
cd ze-ri-skill

# 确保 Node.js 14+ 已安装
node --version

# 赋予执行权限
chmod +x index.js
```

### 基本使用

```bash
# 查询今日吉凶
node index.js

# 查询指定日期
node index.js --date "2026-04-10"

# 查询指定事项的吉日
node index.js --event 婚嫁 --month 2026-05

# 查询指定日期的宜忌
node index.js --date "2026-04-10" --detail

# 输出 JSON 格式
node index.js --date "2026-04-10" --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 日期 (YYYY-MM-DD) | 今日 |
| --event | -e | 事项类型 | 通用 |
| --month | -m | 月份 (YYYY-MM) | 当前月 |
| --detail | --detail | 详细宜忌 | 否 |
| --json | -j | 输出 JSON 格式 | 否 |

### 事项类型

| 类型 | 说明 |
|------|------|
| 婚嫁 | 结婚、订婚、求婚 |
| 开业 | 开店、开张、签约 |
| 动土 | 建房、装修、破土 |
| 出行 | 外出、旅行、迁移 |
| 入宅 | 搬家、入伙 |
| 安葬 | 丧葬、立碑 |
| 祭祀 | 祭祖、拜神 |
| 求医 | 看病、手术 |
| 交易 | 买卖、投资 |
| 通用 | 综合判断 |

## 📚 择日学基础知识

### 建除十二神

| 建除 | 吉凶 | 含义 | 宜 | 忌 |
|------|------|------|-----|-----|
| 建日 | 吉 | 建立、开始 | 开业、动土 | 出行 |
| 除日 | 吉 | 除去、清除 | 治病、清扫 | 嫁娶 |
| 满日 | 吉 | 圆满、充实 | 嫁娶、开业 | 动土 |
| 平日 | 平 | 平稳、普通 | 一般事项 | 重大决策 |
| 定日 | 吉 | 安定、确定 | 签约、订婚 | 出行 |
| 执日 | 吉 | 执行、坚持 | 捕猎、追讨 | 嫁娶 |
| 破日 | 凶 | 破坏、破裂 | 破土、拆除 | 嫁娶、开业 |
| 危日 | 凶 | 危险、危机 | 谨慎行事 | 重大事项 |
| 成日 | 吉 | 成功、成就 | 嫁娶、开业、动土 | 诉讼 |
| 收日 | 吉 | 收获、收纳 | 收割、收款 | 出行 |
| 开日 | 吉 | 开启、开始 | 开业、开光 | 安葬 |
| 闭日 | 凶 | 闭合、结束 | 安葬、修坟 | 开业、嫁娶 |

## 🐍 JavaScript API 调用

```javascript
const zeri = require('./index.js');

// 查询指定日期
const result = zeri.ze_ri_query({
    date: "2026-04-10",
    event: "婚嫁"
});

// 查询本月吉日
const result = zeri.ze_ri_month({
    month: "2026-05",
    event: "开业"
});

// 格式化输出
console.log(zeri.format_output(result));

// 或直接获取字典数据
console.log(result.宜忌);
console.log(result.神煞);
```
