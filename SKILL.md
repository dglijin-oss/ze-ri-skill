> "差之毫厘，谬以千里。" —— 时间一错，吉凶颠倒。

**每次择日前必须：**

1. **校正当前时间** — 以系统实时时间为准，不可用缓存或旧时间
2. **核实公历日期** — 年、月、日，一一确认
3. **核对农历日期** — 建除十二神以农历月为准
4. **验证节气** — 黄道黑道日以节气定月
5. **独立校验脚本输出** — 脚本若有 bug，人工复核

## 时辰择日详解

### 十二时辰吉凶
每个时辰根据当日干支与十二神将的关系判断吉凶：

| 时辰 | 时间 | 说明 |
|------|------|------|
| 子时 | 23:00-01:00 | 夜半，阴气最盛 |
| 丑时 | 01:00-03:00 | 鸡鸣，阳气初生 |
| 寅时 | 03:00-05:00 | 平旦，阳气渐盛 |
| 卯时 | 05:00-07:00 | 日出，阳气上升 |
| 辰时 | 07:00-09:00 | 食时，阳气正旺 |
| 巳时 | 09:00-11:00 | 隅中，阳气极盛 |
| 午时 | 11:00-13:00 | 日中，阳极阴生 |
| 未时 | 13:00-15:00 | 日昳，阴气渐长 |
| 申时 | 15:00-17:00 | 哺时，阴气上升 |
| 酉时 | 17:00-19:00 | 日入，阴气正盛 |
| 戌时 | 19:00-21:00 | 黄昏，阴气渐盛 |
| 亥时 | 21:00-23:00 | 人定，阴气极盛 |

### 择时原则
- **吉事**（婚嫁/开业/搬家）：选辰、巳、午、未时
- **凶事**（丧葬/拆迁）：选戌、亥、子时
- **中性事**（出行/签约）：选寅、卯、申、酉时
